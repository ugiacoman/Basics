QUESTION 1:
How many parameters does the function divide() take? 

ANSWER: 2 parameters (numerator and denominator)



QUESTION 2:
Does the function divide() return a value? 

ANSWER: Yes, the function divide returns the value of the numerator divided by the denominator.



QUESTION 3:
Is the output correct? If it is not, explain why. 

ANSWER: The function correctly divides the numerator by the denominator, however upon returning to the 
main loop, the result = 0. It does not actually display the output returned by the function divide().
